# 📘 Lab 1 — Préparation de l'Espace de Travail

> Le README complet de ce lab est distribué par le formateur (PDF ou lien).  
> Ce dossier ne contient pas de code à écrire — il sert de point de départ pour les labs suivants.

## ✅ Objectifs

- Créer votre repo depuis le template
- Démarrer votre Codespace
- Configurer l'accès AWS
- Vérifier que `terraform version` et `aws sts get-caller-identity` fonctionnent

## 🔑 Commandes de validation

```bash
terraform version
aws --version
aws sts get-caller-identity
```
